Detailed Description

ChessGame encapsulates a game "session". It can either start from scratch or from a given starting position. It is responsible for tracking all aspects of the game. A non-exhasutive list: The board (owns the pieces, and moves, and more), time controls, all UI components like a move history or captured list, and of course the chess engine.

ChessGame is also responsible for routing applicable events from the UI to the engine, like when a player clicks on a square.
